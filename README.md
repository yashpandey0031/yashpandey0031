## 🧠 AI & Machine Learning

- **[GNSS-CLK](https://github.com/yashpandey0031/GNSS-CLK)** — GRU/LSTM models predicting satellite clock bias on NASA CDDIS data. R² = 0.9987 with trend-residual decomposition, MAE 8.85×10⁻⁹s - and a previously uncharacterized LSTM failure mode on drifting time-series along the way. Published at IEEE IC3SIS 2026. [[demo](https://yashpandey0031.github.io/GNSS-CLK/)]
- **[coral-reefs-nlp](https://github.com/yashpandey0031/coral-reefs-nlp)** — ReefMind: SBERT retrieval vs. an LLM-augmented pipeline over 15,945 QA pairs across 831 coral species. The lightweight one won - 18ms latency, 100% retrieval accuracy, a fraction of the cost. Not the result I expected. [[demo](https://nlp-implementation.streamlit.app/)]
- **[anime-recommendation-system](https://github.com/yashpandey0031/anime-recommendation-system-using-tfIdf-and-cosine-similarity)** — TF-IDF and cosine similarity. No neural nets, still good recs.
- **[image-to-music-convertor](https://github.com/yashpandey0031/image-to-music-convertor-using-python)** — Turns pixel colors into sound. Useless. Delightful.
